## 📋 Descripción del Proyecto

**Ferreteria IA** es un sistema inteligente empresarial que integra 6 componentes de Inteligencia Artificial para automatizar, predecir y optimizar la gestión de un negocio de ferretería. El sistema permite registrar ventas, gestionar inventario, predecir demanda, analizar sentimientos de clientes, generar contenido automático y ejecutar procesos autónomos mediante un agente inteligente.

### 🎯 Objetivos del Proyecto

- Demostrar cómo la Inteligencia Artificial puede transformar los negocios
- Integrar múltiples tecnologías de IA en un solo sistema funcional
- Automatizar procesos empresariales clave
- Proporcionar análisis predictivo para la toma de decisiones
- Mejorar la experiencia del cliente mediante análisis de sentimiento

## 🚀 Componentes de IA Implementados

| # | Componente | Descripción | Estado |
|---|------------|-------------|--------|
| 1 | **Sistema Base** | Gestión de clientes, productos, facturas e inventario | ✅ Completado |
| 2 | **Modelo Predictivo** | Predicción de ventas y productos populares usando Random Forest | ✅ Completado |
| 3 | **DSS (Decision Support)** | Alertas automáticas, recomendaciones de compra y descuentos | ✅ Completado |
| 4 | **Análisis de Sentimiento** | Interpretación de opiniones y comentarios de clientes | ✅ Completado |
| 5 | **IA Generativa** | Generación automática de reportes, resúmenes y correos | ✅ Completado |
| 6 | **IA Agentica** | Agente autónomo que monitorea stock y procesa pedidos | ✅ Completado |


## 🛠️ Requisitos del Sistema

- **Python**: 3.12 o superior
- **MySQL**: 8.0 o superior
- **RAM**: 4GB mínimo (8GB recomendado)
- **Sistema Operativo**: Windows, Linux o macOS
